API Endpoints

POST /api/chat

Generate chat completions using various LLMs like GPT-4, Claude, Gemini, and more.

{
  "model": "openai",
  "messages": [
    {"role": "user", "content": "Hello!"}
  ]
}

POST /api/image

Generate images from text prompts using Flux, DALL-E, and other image models.

{
  "prompt": "A sunset over mountains",
  "model": "flux",
  "width": 1024,
  "height": 1024
}

POST /api/video

Generate videos from text prompts using Veo, Seedance, and other video models.

{
  "prompt": "A cat playing",
  "model": "veo",
  "duration": 4
}

GET /api/models/chat

List all available chat models with their capabilities and descriptions.

curl https://your-app.vercel.app/api/models/chat

GET /api/models/image

List all available image generation models.

curl https://your-app.vercel.app/api/models/image

GET /api/models/video

List all available video generation models.

curl https://your-app.vercel.app/api/models/video

Authentication

Include your API key in the Authorization header:

curl -X POST https://your-app.vercel.app/api/chat \
  -H "Authorization: Bearer cgpt_your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "openai",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
